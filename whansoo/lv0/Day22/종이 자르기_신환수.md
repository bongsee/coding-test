# 🦄[프로그래머스]종이 자르기

### 🧡문제 설명
머쓱이는 큰 종이를 1 x 1 크기로 자르려고 합니다. 예를 들어 2 x 2 크기의 종이를 1 x 1 크기로 자르려면 최소 가위질 세 번이 필요합니다.
***
### 💛제한사항
- 0 < M, N < 100
종이를 겹쳐서 자를 수 없습니다.
### 💙입출력 예 설명
입출력 #1
-가로 2 세로 5인 종이는 가로로 1번 세로로 8번 총 가위질 9번이 필요합니다.

입출력 #2
- 이미 1 * 1 크기이므로 0을 return 합니다.
***
### 💜나의 풀이
```javascript
function solution(M, N) {
    return M * N - 1;
}
```
